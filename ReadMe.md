# A Web Server Application Using Socket Programming

## Installation & Running the server

### Prerequesite

Before you can use the application, make sure to install python version 3

### Steps

To install the application open your terminal/cmd and run the following commands:

```bash
git clone git@github.com:waidfj/web_service.git
```

To run the server, execute the following:

```bash
cd web-server
python3 server.py
```
